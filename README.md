## CDP Portal

### Prerequisites

Have installed:

- Git
- Node 6 or higher

### Installation

1) `git clone git@github.com:makerdao/scd-cdp-portal.git`

2) `cd scd-cdp-portal && npm install`

### Run project

- `npm start`
- go to http://localhost:3000 (default port)

For HW support:
- `HTTPS=true npm start`
- go to https://localhost:3000 (default port)
